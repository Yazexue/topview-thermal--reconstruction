# Changelog

## 0.1.0

- Initial public-facing repository structure for the TOPVIEW thermal reconstruction pipeline.
- Added support for `.nc` and `.npz` patch inputs.
- Added grouped split generation to reduce patch-level leakage risk.
- Added deterministic synthetic cloud generation.
- Added IDW and ordinary kriging baselines.
- Added regression U-Net and conditional diffusion training pipelines.
- Added cloud-only evaluation, thin/thick cloud metrics, and thesis-style result table export.
- Added Colab workflow notebook and reproduction documentation.
